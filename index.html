NotepadDB(Saving and Sharing ideas with One Word)


  • Search for File you saved by Name
  • You can Create New Files below
  • Create New Rich Text File with Simple Words
  • Space and Dot are not allowed
  • File Name Must be Unique

This is a Basic and Useful Web Tool for quick saving and sharing data especially Text and Rich text like WepPages or HTML Pages.
You just need to copy and paste the data into the text area and that's it a dedicated url link is created with your FileName you saved.

Most Interesting thing about this tool is that you can access your file on any device with your FileName anytime.

This Web Tool is created with the help of Node.js, Express and MonogDB.
